Module 1 (Sept 17-28)

The main focus of this module will be learning the basics of HTML and CSS to create a Personal Tech Portfolio webpage, hosted on GitHub. Note: Any material presented in this module is fair game for the quiz.

Intro to HTML and CSS

  1. Get your site online by uploading files to gh-pages branch of your repository. See instructions in Module 0 if you're late joining.
  2. REVIEW HTML. Under "Responsive Web Design > Basic HTML": You should be comfortable with all topics up to and including "Create an ordered List".
  3. REVIEW CSS. Under "Responsive Web Design > Basic CSS": You should be comfortable with all topics up to and including "Size Your Images".
  4. If you're already familiar with these topics, feel free to explore further to make your website look even more professional.
